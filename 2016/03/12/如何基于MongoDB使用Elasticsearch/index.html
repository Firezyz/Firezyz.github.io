正文

你是否曾经想过在程序中做一个搜索部件？在早些时间，你可能会使用Solr，或者在Lucene之上封装自己的搜索服务。但是，自2010年，有一个更简单的方法：Elasticsearch(http://elasticsearch.org/)

Elasticsearch是一个基于Lucene的开源存储引擎。它不仅仅是个搜索引擎；它实际上是一个强调搜索性能的一致性或耐久性的文件存储。这就意味着，在许多应用中，你可以使用Elasticsearch作为你的整个后端。

建立一个食谱搜索引擎

本文中，你将学会如何使用Elasticsearch和AngularJS来创建一个食谱搜索引擎，就像 OpenRecipeSearch.com ，那为什么是食谱呢？

  1. 已经有了OpenRecipes，我们的工作会更简单
  2. 为什么不是菜谱呢？

之所以选择菜谱，因为OpenRecipes是一个获取许多家菜谱网站的数据并能以Json下载下来的开源项目。好处在于，Elasticsearch也是使用Json。当然，在我们将菜谱数据拿下来之前，我们得先安装运行Elasticsearch。

  • 下载Elasticsearch 解压。
  • 打开一个终端，cd 到你刚刚解压的目录，运行 bin/elasticsearch（Windows上是bin/elasticsearch.bat ）,这样你就已经启动你自己的elasticsearch程序了。

Elasticsearch其中最好的一点是它的RESTful后端，这使得他能适用于各种环境。我们将要使用JavaScript环境，但是你可以使用 其他任何一种；代码也会看起开比较简单。如果你愿意，你可以参看这个简要参考。

建立一个菜谱搜索界面

这就是前面为什么说要使用Angular。我选择Angular有两个原因：一者是因为我想用，再者是Elasticsearch的JavaScript库提供了强大的Angular适配。我会把设计作为读者的练习，但我会告诉你的HTML的重要部分。

现在集中注意力到Elasticsearch和Angular。我推荐使用Tower，但是你也可以选择直接现在它们。打开你的index.html文件，然后在你常放JavaScript的地方插入以下代码：

<script src="path/to/angular/angular.js"></script>
<script src="path/to/elasticsearch/elasticsearch.angular.js"></script>

现在让我们先暂停一下，思考一下我们的app将要怎么运行：

  1. 用户输入查询词。
  2. 我们将查询词作为搜索内容发送给Elasticsearch。
  3. 我们取回结果。
  4. 然后将结果展示给用户。

下面放的主要是我们的搜索引擎的嵌入了Angular的核心HTML代码。如果你从来没有使用过Angular也没关系。领会下面代码你只需要知道以下几点：

  1. HTML中以ng开头的属性就是Angular指令。
  2. 在ng-app和ng-controller包裹之中的Angular指令才是有效的。ng-app和ng-controller不用放在同一个元素上，但是可以放在同一元素。
  3. HTML中所有其他值的引用变量都能在JavaScript的$scope对象中看到。

  4. 在”{ { } }”包裹的部分是模板变量。

现在，我们可以开始写JavaScript了。首先我们先写一个模块叫做MyOpenRecipes。

/**
 * Create the module. Set it up to use html5 mode.
 */
window.MyOpenRecipes = angular.module('myOpenRecipes', ['elasticsearch'],
    ['$locationProvider', function($locationProvider) {
        $locationProvider.html5Mode(true);
    }]
);

对AngularJS新手来说，['$locationProvider', function($locationProvider) {...}]是为了告诉Angular我们需要把$locationProvider传到handler方法中，所以我们可以使用它。

接下来，我们开始写controller，命名为recipeCtrl。

/**
 * Create a controller to interact with the UI.
 */
MyOpenRecipes.controller('recipeCtrl', ['recipeService', '$scope', '$location', function(recipes, $scope, $location) {
  // Provide some nice initial choices
  var initChoices = [
      "rendang",
      "nasi goreng",
      "pad thai",
      "pizza",
      "lasagne",
      "ice cream",
      "schnitzel",
      "hummous"
  ];
  var idx = Math.floor(Math.random() * initChoices.length);
  // Initialize the scope defaults.
  $scope.recipes = [];        // An array of recipe results to display
  $scope.page = 0;            // A counter to keep track of our current page
  $scope.allResults = false;  // Whether or not all results have been found.
  // And, a random search term to start if none was present on page load.
  $scope.searchTerm = $location.search().q || initChoices[idx];
  /**
   * A fresh search. Reset the scope variables to their defaults, set
   * the q query parameter, and load more results.
   */
  $scope.search = function() {
    $scope.page = 0;
    $scope.recipes = [];
    $scope.allResults = false;
    $location.search({'q': $scope.searchTerm});
    $scope.loadMore();
  };
  /**
   * Load the next page of results, incrementing the page counter.
   * When query is finished, push results onto $scope.recipes and decide
   * whether all results have been returned (i.e. were 10 results returned?)
   */
  $scope.loadMore = function() {
    recipes.search($scope.searchTerm, $scope.page++).then(function(results) {
      if (results.length !== 10) {
        $scope.allResults = true;
      }
  var ii = 0;
  for (; ii < results.length; ii++) {
        $scope.recipes.push(results[ii]);
      }
    });
  };
  // Load results on first run
  $scope.loadMore();
}]);

MyOpenRecipes.factory('recipeService', ['$q', 'esFactory', '$location', function($q, elasticsearch, $location) {
  var client = elasticsearch({
    host: $location.host() + ':9200'
  });
  /**
   * Given a term and an offset, load another round of 10 recipes.
   *
   * Returns a promise.
   */
  var search = function(term, offset) {
    var deferred = $q.defer();
    var query = {
      match: {
        _all: term
      }
    };
    client.search({
      index: 'recipes',
      type: 'recipe',
      body: {
        size: 10,
        from: (offset || 0) * 10,
        query: query
      }
    }).then(function(result) {
      var ii = 0, hits_in, hits_out = [];
      hits_in = (result.hits || {}).hits || [];
      for(; ii < hits_in.length; ii++) {
        hits_out.push(hits_in[ii]._source);
      }
      deferred.resolve(hits_out);
    }, deferred.reject);
    return deferred.promise;
  };
  // Since this is a factory method, we return an object representing the actual service.
  return {
    search: search
  };
}]);
