解决ubuntu14.04 LTS误删network manager导致网络图标消失不能上网

发表于 2016-03-11   |   分类于 鼓捣   |  

起因

因为手贱卸载软件时误删了network namager，导致网络图标消失了，不能上网，打开系统设置>网络,也出现网络服务与当前网络管理器不兼容的错误，今天总算完美解决这一问题：

方案一：

首先试试这个吧：

如果依次敲下以上代码就可以搞定的话，那么恭喜你，就不用往下看了；

方案二：

如果不行的话：

手动设置固定ip

保存，重启吧

设置DNS

你会注意到该文件有这么一句话：

#DO NOT EDIT THIS FILE BY HAND – YOUR CHANGES WILL BE OVERWRITTEN

就是说你手动该会被覆盖，先不管他
在文件结束加上
nameserver 192.168.0.1（你的DNS）
保存，记住不要重启，一旦重启，就会被重写。
然后不出意外，你已经可以上网了，虽然右上角还是没有图标，但是已经可以打开网页了

安装network manager

重启即可看到网络图标

然后将1步骤中添加的内容删除，通过网络图标设置你的网络即可（不删也不要紧，只是网络管理器里显示你的网卡是托管状态）

最后

如果你在此教程中出现其他问题：
移步：
